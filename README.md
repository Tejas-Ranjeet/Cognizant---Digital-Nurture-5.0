## How to Run

### Java Projects

Compile and run using any Java IDE such as:
- VS Code
- IntelliJ IDEA
- Eclipse

### Maven Projects

Open the project containing `pom.xml` and run:

```bash
mvn clean compile
mvn exec:java
```
